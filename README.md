# Project Overview
-	Designed normalized relational database schema (Users, Songs, Genres, Moods, Devices, Listening History)
-	Implemented primary and foreign key relationships to maintain data integrity
-	Wrote complex SQL queries using JOIN, GROUP BY, ORDER BY, HAVING, aggregate functions
-	Analysed most popular songs, artists, and genres
-	Identified active listeners and peak listening time
-	Analysed device usage patterns (mobile, desktop, etc.)
-	Performed region-wise and mood-based listening analysis
-	Detected repeated listening behaviour and user engagement patterns
-	Insights: Analysed music listening data to identify popular genres, active users, peak listening time, and user behaviour patterns.
